### Superdiff
![NPM Downloads](https://img.shields.io/npm/dy/%40donedeal0%2Fsuperdiff?logo=npm)
![GitHub Repo stars](https://img.shields.io/github/stars/DoneDeal0/superdiff)

Superdiff provides a complete and readable diff for both arrays and objects. Plus, it supports stream and file inputs for handling large datasets efficiently, is battle-tested, has zero dependencies, and is super fast. It has received github stars from engineers at Linkedin, AWS, ByteDance, Shopify, Github, Yahoo, Alibaba, Vodafone, Meta Open Source, CodePen, etc.
